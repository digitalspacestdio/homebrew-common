# Homebrew Common Tap
Homebrew/Linuxbrew common formulas

### Add new tap
```
brew tap digitalspacestdio/common
```

### Mysql Servers
```
brew install mysql57
brew install mysql80
brew install mysql84
```

### Imagemagick 6
```
brew install imagemagick6
```

### OpenSSL 1.1
```
brew install openssl@1.1
```
